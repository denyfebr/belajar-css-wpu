My Projects

SMART SCAN (Supply Chain Assessment Navigator) is a platform developed with ASP.NET MVC C# and used to obtain comprehensive profile information of suppliers who produce Crude Palm Oil (CPO) and Palm Kernel (PK), which are subsequently supplied to GAR (Golden Agri-Resources) Refineries.

Through SMART SCAN, users can access detailed supplier profiles, including supply volume records, track records, capacity building records, contact person information, and geolocation data.

Developed a web-based Online Supplier Self-Assessment (SSA) Form for the Compliances Team. The application was built using the PHP CodeIgniter framework.

The Online Supplier Self-Assessment (SSA) system, named SMART CHECK, provides assessment tools for suppliers and generates compliance scores based on their responses.

SMART Learning is an e-learning platform built with Moodle framework designed to provide educational materials that facilitate knowledge transfer on sustainability practices to GAR’s suppliers.

Developed a data visualization dashboard using Power BI, which compiles and integrates data from multiple application databases into a centralized and interactive reporting platform.
